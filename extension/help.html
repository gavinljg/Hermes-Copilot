Hermes Copilot

Use the current browser page with your local Hermes setup for summaries, follow-up questions, and page-aware assistance. The extension reads and displays context; model calls still go through your own Hermes configuration.

把浏览器当前页面交给你本机的 Hermes 来理解、总结和追问。扩展只负责读取页面与展示对话，模型调用仍走你自己的 Hermes 配置。

1. Install the local Bridge

The Bridge runs on your computer, listens on 127.0.0.1:18765 by default, and forwards extension requests to Hermes.

curl -fsSL https://raw.githubusercontent.com/gavinljg/Hermes-Copilot/main/install.sh | bash

2. Load the extension

  1. Open edge://extensions/.
  2. Turn on Developer mode.
  3. Click Load unpacked.
  4. Select ~/.local/share/hermes-copilot/extension.

3. Start using it

  • Open any web page and click the extension icon to open the side panel.
  • Click “Summarize page”, or type your own question.
  • Open “Advanced Settings” when you need to switch models.
  • Open “Chat History” to restore previous conversations.

4. Common fixes

  • If the Bridge is not connected, run the install command and click Recheck.
  • If a site cannot be read, make sure the extension is allowed to access that site.
  • Browser internal pages usually cannot expose body text; only title and URL may be available.

Where your data goes

The extension reads the current tab title, URL, visible text, selected text, form summaries, and resource summaries, then sends them to the local Bridge. The Bridge calls your Hermes configuration; if Hermes is configured with a cloud model provider, that provider may receive the page content.

1. 安装本地 Bridge

Bridge 运行在你的电脑上，默认监听 127.0.0.1:18765，负责把扩展请求转给 Hermes。

curl -fsSL https://raw.githubusercontent.com/gavinljg/Hermes-Copilot/main/install.sh | bash

2. 加载扩展

  1. 打开 edge://extensions/。
  2. 打开开发者模式。
  3. 点击加载解压缩的扩展。
  4. 选择 ~/.local/share/hermes-copilot/extension。

3. 开始使用

  • 打开任意网页后点扩展图标，侧边栏会读取当前页面。
  • 点“总结页面”，或直接输入你的问题。
  • 需要切换模型时，展开“高级设置”。
  • 想延续之前的问题，展开“会话历史”并恢复会话。

4. 常见处理

  • 如果提示未连接 Bridge，先确认安装脚本执行完成，再点“重新检测”。
  • 如果某些站点无法读取，请确认扩展权限已允许访问该站点。
  • 浏览器内部页面通常无法读取正文，只能使用标题和 URL。

数据会去哪里

扩展会读取当前标签页的标题、URL、可见正文、选中文本、表单摘要和资源请求摘要，并发送给本机 Bridge。Bridge 再调用你的 Hermes 配置；如果 Hermes 配置了云端模型提供商，该提供商可能会收到这些页面内容。

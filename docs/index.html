Hermes Copilot

一个连接 Microsoft Edge 当前页面与本机 Hermes 的浏览器侧边栏助手。

安装 Bridge

Bridge 会在你的电脑上运行，只连接本机 Hermes。运行前可以先查看脚本源码。

curl -fsSL https://raw.githubusercontent.com/gavinljg/Hermes-Copilot/main/install.sh | bash

加载扩展

  1. 打开 edge://extensions/
  2. 启用 Developer mode
  3. 选择 Load unpacked
  4. 选择 ~/.local/share/hermes-copilot/extension

数据说明

扩展会读取当前标签页的标题、URL、可见正文、选中文本、表单摘要和资源请求摘要，然后发送给本机 Bridge。Bridge 调用你本机 Hermes 的模型配置。项目本身不提供云端服务，也不保存你的页面内容。
